# Images – Project Visuals

This folder contains visual assets used in the Atliq Hardware SQL analysis project.

## Included Visuals

• Product Growth Chart – shows 36% increase in product portfolio  
• Segment Distribution Chart – shows product distribution across segments  
• Channel Contribution Chart – shows revenue share by sales channel  
• SQL Query Screenshot – sample query used in analysis  
• Presentation Preview – snapshot of final presentation  

## Purpose

These visuals help communicate insights more effectively and make the project easier to understand for business stakeholders and recruiters.
