# Dataset – Atliq Hardware

## Overview

This folder contains information about the dataset used in the SQL analysis project.

The dataset is part of the Codebasics SQL Resume Challenge and represents a fictional company called Atliq Hardware.

---

## Contents

• dataset_link.txt – link to access dataset  
• data_description.md – explanation of tables and structure  

---

## Note

The dataset is not uploaded directly to this repository to keep it lightweight and maintain best practices.

Instead, a reference link is provided.

---

## Purpose

This dataset enables analysis of:

• Sales performance  
• Product growth  
• Customer behavior  
• Pricing and discount strategies
